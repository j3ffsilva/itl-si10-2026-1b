## DESCRIÇÃO

Neste encontro, você irá praticar a geração dados sintéticos que reflitam o comportamento do consumidor com regras e distribuições estatísticas.

Objetivos de aprendizagem
- OA01: Explicar quando dados sintéticos são necessários em projetos com restrição de acesso a dados.
- OA02: Gerar dados sintéticos com base em regras e distribuições estatísticas do problema.
- OA03: Julgar se a base sintética produzida atende ao contexto de análise comportamental.
